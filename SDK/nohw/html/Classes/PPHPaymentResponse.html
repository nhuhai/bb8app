isPaymentAuthorization

Whether this was an authorization rather than a sale. In this context, authorization means a “complete transaction and verification of funds” that provides a token that can be captured at a future time. Note that this is NOT the same as an “EMV authorization” which is a part of the transaction flow for EMV.

@property (nonatomic, assign) BOOL isPaymentAuthorization
