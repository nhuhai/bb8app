PPHCardReaderMetadata Class Reference

Inherits from NSObject
Conforms to NSCoding
Declared in PPHCardReaderMetadata.h

Overview

PPHCardReaderMetadata represents what can be determined about a connected card reader device without actually connecting or talking to it.
