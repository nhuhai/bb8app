PPHLoggingDelegate Protocol Reference

Conforms to NSObject
Declared in PPHLoggingDelegate.h

Overview

For debugging purposes it can be helpful to see our internal logs. Implement this protocol and set the logging delegate on PayPalHereSDK. You should generally not set this delegate in production code as it will have a theoretical performance impact (we’ll call respondsToSelector a lot). If you respond to a given selector, we’ll put the message together, which will cause a bunch of string munging to happen. So basically, use this when you mean it, not just on a whim.
